# Web Scraping using Python

## Project Description
This project extracts book details from a website using Python Web Scraping techniques and stores the data in CSV format.

## Tools Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Dataset
Data collected from:
https://books.toscrape.com/

## Analysis Performed
- Sent HTTP requests to website
- Parsed HTML content using BeautifulSoup
- Extracted Book Title, Price, and Rating
- Stored extracted data in CSV format

## Output Files
- books_data.csv
- quotes.csv
- scraper.py
- Task1_Report.pdf
